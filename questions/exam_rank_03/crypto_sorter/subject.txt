# Custom String Sort with Multiple Keys

## Assignment name

crypto_sorter

## Expected files

crypto_sorter.py

## Function

def crypto_sorter(words: list[str]) -> list[str]:

Given a list of strings `words`, sort them based on the following rules:

1. Sort by length of the string in ascending order.
2. If two strings have the same length, sort by ASCII lexicographical order.
3. If still tied, sort by the number of vowels in ascending order.

The function must return the sorted list.

## Examples

Example 1:

Input: words = ["apple", "Banana", "kiwi", "orange", "Egg"]
Output: ["Egg", "kiwi", "apple", "Banana", "orange"]

Example 2:

Input: words = ["a", "bb", "ccc", "dd"]
Output: ["a", "bb", "dd", "ccc"]

Example 3:

Input: words = ["abc", "acb", "bac"]
Output: ["abc", "acb", "bac"]
